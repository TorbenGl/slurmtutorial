# Figures

Drop overview images for the tutorial here, then reference them from the main
[`../Readme.md`](../Readme.md) (see its "Including Figures" section). Suggested
files:

- `cluster-overview.png` — login node vs. compute nodes vs. controller
- `job-lifecycle.png` — submit → pending → running → completed
- `pipeline.png` — job-dependency / multi-stage workflow

**Tips:** use PNG for screenshots, SVG for diagrams (stays crisp). Keep paths
relative so the repo works for everyone.

Include in markdown:

```markdown
![Cluster overview](figures/cluster-overview.png)
```
